# 🧙‍♂️ Descubra Sua Classe de RPG (qualSuaClasse)

Bem-vindo ao **qualSuaClasse**! Este é um aplicativo web interativo e divertido onde os usuários respondem a um quiz de 10 perguntas de personalidade e comportamento para descobrir qual classe de RPG clássica ou de fantasia mais combina com o seu estilo de jogo.

O projeto conta com um algoritmo simples de pontuação acumulativa que avalia as respostas em tempo real e, ao final, revela uma das **15 classes disponíveis**, acompanhada de uma descrição detalhada.

---

## 🚀 Funcionalidades

- **10 Perguntas Temáticas:** Situações de combate, exploração de masmorras, escolha de armas e dilemas morais.
- **15 Classes de RPG:** Desde os clássicos Guerreiro, Mago e Ladino, até escolhas mais ricas como Necromante, Alquimista, Invocador, Elfo (Arqueiro/Guardião), Bárbaro e Anão.
- **Acúmulo de Pontos Dinâmico:** Cada resposta soma pontos discretos para grupos de classes correspondentes.
- **Interface Intuitiva e Responsiva:** Visual limpo e transições suaves focado na experiência do usuário.
- **Resultado Instantâneo:** Mostra o veredito final com a lore/descrição da classe vencedora e opção de reiniciar o quiz.
